# 1️⃣ 학습 내용 정리

## 1. 자료구조와 순차 자료형

### 개념정리

3.1.1.튜플
튜플은 한번 할당되면 변경할 수 없는 고정 길이를 갖는 파이썬의 순차 자료형. 가장 쉬운 생성 방법= 쉼표로 구분되는 일련의 값을 괄호로 감싸기 
만약 튜플같은 표현의 변수에 튜플 대입 시 파이썬은 등호 오른쪽에 있는 변수에서 값을 분리함 



3.1.2 리스트 
튜플과 대조적으로 리스트는 크기, 내용 변경 가능 
list 함수는 이터레이터나 제너레이터 표현에서 실젯값을 모두 담기 위함 
append메서드, insert 메서드로 새로운 값을 추가할 수 있다 
+연산자로 두개의 리스트를 합칠 수 있다 
sort함수를 이용해 리스트를 정렬할 수 있다 
슬라이싱: 리스트와 같은 자료형은 색인 연산자 안에 start:stop을 지정해 원하는 크기만큼 잘라낼 수 있다 


3.1.3 딕셔너리 
딕셔너리는 파이썬 내장 자료구조중에서 가장 중요한 자료구조이다 
각 키는 값과 연관되어 특정 키가 주어지면 값을 편리하게 검색, 삽입, 수정 또는 삭제가 가능하다. 

3.1.4 집합 
고유한 원소만 담는 정렬되지 않은 자료형 
집합 연산- add, clear, remove, pop, union, update, intersection,difference등 

3.1.5 내장 순차 자료형 함수 
순차 자료형에 사요할 수 있는 함수 
- enumerate
-sorted 
-zip : 여러개의 리스트나 튜플 또는 다른 순차 자료형을 서로 짝지어 튜플 리스트 생성 
-reversed: 순차 자료형 역순 순회 


3.1.6 리스트, 집합, 딕셔너리 표기법 
리스트 표기법은 간결한 표현으로 새로운 리스트를 만들 수 있음
